DownloadSFTP
=====

Realiza backups através de sftp de uma lista inserida no banco de dados.

## Libs necessárias ##

* paramiko
* flask
* procname
* psycopg2
* pidmon
* psutil

### Como usar ###

Inicie primeiro o server.py

```$ python server.py```

Após o servidor inciado, vá até o navegador e entre no link http://localhost:5000/manager/start
